Acerca del Proyecto

Este proyecto fue desarrollado con Flask como parte de la tarea de la semana 11 de Desarrollo de Aplicaciones Web.

El sistema implementa Programación Orientada a Objetos, uso de colecciones como listas y diccionarios, conexión a base de datos SQLite y operaciones CRUD reales para la gestión del inventario de libros.

Tecnologías usadas

  • Python
  • Flask
  • SQLite
  • HTML y CSS

Características del sistema

  • Agregar libros
  • Buscar libros
  • Actualizar cantidad y precio
  • Eliminar libros
  • Mostrar inventario completo
